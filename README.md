# CommandPermission
Configure for all commands to have a specific permission


Config.yml
![Quote]
Comandos:
  give:
    Sem_Perm: "&cYou dont have permission"
    Perms:
      - "staff.pay"
      - "naosei.p"
  op:
    Sem_Perm: "&cYou dont have permission"
    Perms:
      - "staff.pay"
